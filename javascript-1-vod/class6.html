Iteration aka Looping

Looping

  • Another common behavior of code is to repeat tasks--also known as looping or iterating
  • JavaScript has multiple ways of executing loops
Loops and Iteration (MDN)

Standard for Loop

  • A for loop is often the first type of loop we learn in JavaScript
  • A for loop executes a given task a set number of times and under specific conditions

for Loop Syntax

for (expression1; expression2; expression3) {

  // code to be repeated

}

// expression1 starts or initializes the loop

// expression2 is the condition that 
// determines if/when the loop should run

// expression3 counts the looping cycle

for Loop Syntax

for ( let i = 1; i < 5; i++ ){

  console.log(`I love JavaScript`)

}
// I love JavaScript
// I love JavaScript
// I love JavaScript
// I love JavaScript
  • let i = 1; initializes a variable i that starts the loop
  • i < 5; how long the loop runs as long as i is less than 5
  • i++ keeps track or count of the looping cycle

Decrementing Loops

Loops can also count down by:

  • Changing the initializing value
  • Changing the condition to run against
  • Using the -- decrement operator
for (let dirtySocks = 10; dirtySocks > 0; dirtySocks--) {

  console.log("Dirty socks remaining: " + dirtysocks);  

}

// Dirty socks remaining: 10
// Dirty socks remaining: 9
// ...
// Dirty socks remaining: 2
// Dirty socks remaining: 1

Looping through Strings

We can loop through characters of a string

let fruit = 'banana'

for (let i = 0; i <= fruit.length; i++){

  console.log(fruit[i])
  
}
// b
// a
// n
// a
// n
// a

Loops & Conditionals

Loops can execute conditionals

for(let num = 0; num <= 10; num++){

  if(num % 2 === 0 ){
  
    console.log('Lauren')
  
  } else {
  
    console.log(num)
  
  }

}
// If num is even, print the name 'Lauren'
// Else if num is odd, just print the value of num

Infinite Loops

Loops are finicky!

  • When written incorrectly or inefficiently, a loop will crash your browser or application
  • Usually this happens when we write a loop in which the condition is always true
  • This means the loop will never stop running

Infinite Loops

// This loop will run indefinitely because
// it will ALWAYS be true that num is not 
// equal to super.

for(let num = 0; num !== 'super'; num++){

console.log('Lauren ' + num)

}

while Loop

  • Similar to a for loop, a while loop also checks that a specific condition must be met to run or execute code
  • It has a different syntax, but the concept is the same

while Loop Syntax

// Set up a starting condition
// Then the while loop:

while (condition is true) {

// code to run

// counter (increment or decrement)

}

while Loop Syntax

// Starting condition:
let numOfBooks = 1

// The loop
while (numOfBooks <= 10) {

// Code to run
console.log(numOfBooks)

// Counter
numOfBooks++

}

Decrementing a while Loop

Remember that to decrement, we must update the condition and the counter

let applesEaten = 10

while (applesEaten >= 0){

console.log(applesEaten);

applesEaten--;

}

for or while

Both loops do the same thing; but if deciding between the two:

  • Use a for loop if you know how many times the loop should run
  • Use a while loop if you want the loop to run based on a condition rather than a set number of times
