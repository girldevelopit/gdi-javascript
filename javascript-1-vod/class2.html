Numbers

Integers and decimals make up the number data type

let age = 35

let taxRate = 27.5

Math Operators

We can use standard math operators with the number type

let sum = 35 + 20 // Addition

let difference = 200 - 130 // Subtraction

let product = 10 * 50 // Multiplication

let average = 18 / 4 // Division

let remainder = 18 % 3 // Modulo

/* Modulo is a special operator that returns
the remainder of a division */
Note: Calculations in JavaScript follow the PEMDAS rule.

Math Operators: Variables

We can also perform math using number variables

let currentYear = 2022

let birthYear = 1993

let age = currentYear - birthYear

console.log(age)

Math Operators: Incrementing

The increment operator ++ adds one to a value

let age = 2

age++;

console.log(age) // This should be 3

let newAge = ++age;

console.log(newAge) // This should be 4

Math Operators: Decrementing

The decrement operator -- subtracts one from a value

let year = 1992

year--;

console.log(year) // This should be 3

let oldYear = --year;

console.log(oldYear) // This should be 4

Math Operators: Compound Assignments

  • Compound assignment operators combine an assignment and a math operator
  • They take the value to the right of the assignment operator and apply it to the variable
let tip = 15

console.log( tip += 20 ) // 35

Math Operators: Compound Assignments

The main math operators can be turned into compound operators

let tip = 15

console.log( tip += 20 ) // 35

console.log( tip -= 10 ) // 25

console.log( tip *= 4 ) // 100

console.log( tip /= 8 ) // 12.5

Strings and Compound Operators

  • Strings can be combined using the += operator
  • Using other operators will return NaN: Not a Number
let greeting = "hello"

greeting += " world"

console.log(greeting) // hello world

greeting -= " world"

console.log(greeting) // NaN

Combining Strings and Numbers

  • Strings and numbers can be combined with + operator
console.log( 7 + " oranges" ) // 7 oranges

let myAge = "I am "

console.log(myAge += 8) // I am 8
  • This is not recommended; turn the number into a string data type instead

Numbers as Strings

  • Numbers can be written as strings by adding quote marks
  • But calculations done this way are unreliable and can introduce bugs
let num1 = '5' // Number as a String

console.log (num1 + 5) // 55

console.log (num1 * 5) // 25

console.log (num1 - 5) // 0

Checking Data Types

typeof Operator

To make sure we're working with the correct data types, use the typeof operator for verification; this can prevent bugs

let animal = 'dog'
console.log(typeof animal) // string

console.log(typeof 7) // number
console.log(typeof '7') // string

console.log(true) // boolean

String.length

Use the length property to find the length of a string

let badPassword = 'mydob'

console.log(badPassword.length) // 5

Zero Index

  • JavaScript is a zero-indexed language
  • We count from 0, not 1
  • Every character in a string has an index value--a number corresponding to its position in the string

Accessing String Characters

Type the string variable and use the bracket notation []:

let badPassword = 'mydob'

console.log( badPassword[0] ) // 'm'

console.log( badPassword[1] ) // 'y'

console.log( badPassword[2] ) // 'd'

console.log( badPassword[3] ) // 'o'

console.log( badPassword[4] ) // 'b'

string.indexOf()

We can search for characters

let shortName = 'Mary'
console.log(shortName.indexOf('a')) // 1
// Returns index of first occurrence of 'a'

let song = "99 bottles of wine on the wall..."
console.log(song.indexOf('beer')) // -1

string.toUpperCase()

We can change case

let greeting = 'hello world'

let shouting = greeting.toUpperCase()

console.log(shouting) // HELLO WORLD

string.replace()

We can replace characters

let statement = 'I Love JavaScript!'

// Replace words
statement = statement.replace('JavaScript', 'Python')
console.log(statement)

// Replace characters
statement = statement.replace('o', '0')
console.log(statement)
