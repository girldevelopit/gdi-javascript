Boolean

The boolean data type has two values true and false

let catsAreTheBest = true;

let dogsAdoreCats = false;

Default Boolean Values

All variables in JavaScript have a default Boolean value:

They're either "truthy" or "falsy"

Truthy and Falsy Boolean Values

    Truthy
  • true
  • 4
  • 'hello'
  • '0'
  • []
  • {}
    Falsy
  • false
  • 0
  • -0
  • ' '
  • null
  • undefined

Boolean Function

We can use the Boolean function to determine
if a value is truthy or falsy

Boolean('Jackie') // true

Boolean (-0) // false

Boolean (5 < 3) // false

Boolean (undefined) // false

Comparison Operators

Comparison operators compare two values
and return a boolean (true or false)

  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to
console.log( 19 > 20 ) // false

console.log( 5 >= 5 ) // true

console.log( 0 < 2 ) // true

console.log( 6 <= 8 ) // true
JavaScript Comparison and Logical Operators (W3Schools)

Equality Operators

Equality operators test if two values are the same. The equality operators look very similar but work differently.

Loose Equality

Double equals '==' or 'loose equality'
evaluates if values are the same

console.log( 5 == '5' ) // true
JavaScript — Double Equals vs. Triple Equals (Codeburst)

Strict Equality

Triple equals '===' or 'strict equality'
evaluates value AND type

console.log( 5 === '5' ) // false

// 5 as a number is not the same as 5 as a string value

To prevent bugs, use strict equality over loose equality

JavaScript — Double Equals vs. Triple Equals (Codeburst)

Inequality

Both equality operators have their negation counterparts:

  • Loose inequality
console.log( 5 != '5' ) // false
  • Strict inequality
console.log( 5 !== '5' ) // true

Conditionals

Conditional statements control the behavior of our application. They determine the conditions under which code should run, or in what way it should be executed.

if Statement

If a condition meets a criteria, then run the code

if(condition is true){

  then do something...
}

let catsSay = 'meow';

if(catsSay === 'meow'){

  console.log(`I'm a cat and I love to sleep all day`)
}

if-else Statement

if-else extends the if statement by taking into account what should happen if the original condition is NOT met

if(condition is true){
  then do a...
} else {
  do b...
}

let catsSay = 'meow';

if(catsSay === 'meow'){

  console.log(`I'm a cat and I love to sleep all day`)

} else {
  
  console.log(`I must be a dog then.... Woof!`)

}

Ternary Operator

With ES6, we have a new, shorter way to write
conditionals with the ternary operator statement

// ternary statement synax:

condition we're evaluating ? what to do if true : what to do if false

let catsSay = 'meow';

let catResponse = `I'm a cat and I love to sleep all day`

let dogResponse = `I must be a dog then.... Woof!`

catsSay === 'meow' ? console.log(catResponse) : console.log(dogResponse)

Conditional Chains

We can create conditional chains to
evaluate more than two conditions

if ( condition1 is true ) {

  do a...

} else if ( a different condition is true ) {

  do b...

} else if ( another condition is true ) {

  do c...

} 

...

} else {

  None of the above conditions is true? Then do z...
  and let's end this chain!

}

Chaining Conditionals

let today = 'thursday';

if( today === 'monday' ){

  console.log( 'Here we go again....' )

} else if( today === 'tuesday' ){

  console.log( 'Are we done yet?' )

} else if( today === 'wednesday' ){
  
  console.log( 'Getting a bit excited...' )

} else {

  console.log( 'Weekend is almost here!' )

}

Logical Operators

Logical operators check the logic between values or
conditions. There are three operators:

  • AND &&
  • OR ||
  • NOT !

JavaScript Comparison and Logical Operators (Programiz)

&& : The AND Operator

The AND operator returns true if two conditions being evaluated are both true; otherwise it returns false

let sandyAge = 7

let jamalAge = 5

console.log( sandyAge > 4 && jamalAge < 9 ) // true

console.log( sandyAge > 4 && jamalAge > 9 ) // false

|| : The OR Operator

The OR operator returns true if one condition is true;
it returns false when both are false

let sandyAge = 7

let jamalAge = 5

console.log( sandyAge > 4 || jamalAge > 9 ) // true

console.log( sandyAge < 4 || jamalAge < 9 ) // false

! : The NOT Operator

The NOT operator returns the opposite condition of what's evaluated. If a value is true, the operator will return false and vice versa

let sandyAge = 7

let jamalAge = 5

console.log( !sandyAge ) // false

console.log( !jamalAge > 3 ) // false

!(true) // false

!(false) // true
