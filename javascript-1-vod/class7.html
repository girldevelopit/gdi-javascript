Arrays

Arrays

  • Arrays are another JavaScript data type
  • An array is an ordered list of values
  • Arrays are great for storing multiple values under a single variable name
let fruits = ['apple', 'pear', 'mango']
Arrays (MDN)

Array Syntax

An array is declared with a variable name and square
brackets [] that contain values separated by commas

 let arrayName = [ item1, item2, item3, item4, .... ]
A Beginner's Guide to JavaScript Arrays (JavaScriptTutorial.net)

Arrays

Arrays can hold different types of data, even other arrays!

let primaryColors = ['red', 'blue', 'green']

let lotteryNumbers = [2, 24, 54, 9, 30, 44]

let randomValues = ['hello', 66, true, ['item1', 'item2'] ]

Zero Index

Arrays are zero-indexed, just like strings;
we count from 0, not 1

let namesList = ['Anita', 'Naoko', 'Olga', 'Sandra']

console.log(namesList[0]) // Prints Anita
// Anita is the first item, and has an index value of 0

console.log(namesList[3]) // Prints Sandra
// Sandra is the fourth item, but has an index value of 3
// because we're counting from 0

Updating an Array

Knowing the indices or positions in an array, means we can manually add or overwrite the values in the array

Manually Add a New Item 

 // namesList has 4 names, let's add a 5th

let namesList = ['Anita', 'Naoko', 'Olga', 'Sandra']

namesList[4] = 'Jules'

console.log(namesList)

Manually Overwrite an Item 

 // Our list: 'Anita', 'Naoko', 'Olga', 'Sandra', 'Jules'

// Let's update Olga to Kiki

namesList[2] = 'Kiki'

console.log(namesList)

Array Methods

JavaScript has default array methods that
allow us to work with the data in an array

Array.length

We can use the length property to count
the number of items in an array

 let countries = ['Peru', 'Korea', 'Canada', 'Ghana']

console.log(countries.length) // 4
Array.prototype.length (MDN)

Adding to the back of an array
.push()

let countries = ['Peru', 'Korea', 'Canada', 'Ghana']

countries.push('Belgium') // 5

Note: .push() returns the new
length of the array, not the array itself

Array.prototype.push() (MDN)

Adding to the front of an array
.unshift()

// Still working with the countries array...

countries.unshift('Turkey') // 6

console.log(countries)

// ['Turkey', 'Peru', 'Korea', 'Canada', 'Ghana', 'Belgium']

Note: .unshift() also returns the new
length of the array, not the array itself

Array.prototype.unshift() (MDN)

Looping Through Arrays

There are a number of ways to iterate through
an array; let's explore two basic approaches:

  • for loop
  • for of loop

Using a Basic for Loop

  • The for loop syntax remains unchanged
  • But we use the array's length as the condition that controls the loop
let oneNameSingers = ['Sia', 'Lizzo', 'Beyonce', 'Pink' ];
    
    let arrLength = oneNameSingers.length;
      
    for(let i=0; i < arrLength; i++){
    
      console.log(oneNameSingers[i])
    
    }

Using the for of Loop

  • The for of was introduced in ES6
  • It can be used on any data type that's countainable, such as arrays and strings
  • It has a different syntax:
for (variable of iterable) {
      
      // ...
      // usually returning or doing something with 'variable'
    
    }

iterable is the object that's being looped
variable is each item in the object

for of Loop

  • With the for of loop, we don't need to rely
    on the array's length
let moreOneNameCelebs = ['Oprah', 'Bjork', 'Madonna', 'JayZ']
    
    for( celeb of moreOneNameCelebs ){
    
      console.log(celeb)
    
    }
