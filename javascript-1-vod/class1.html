Values and Variables

Value: A basic unit of data or information

Variable: A "container" to store information

Declaring a Variable

There are three keywords used to declare variables in Javascript:

let firstName;
  
const userName;

var yourName;

Initializing a Variable

let firstName = Sara;

We initialize a variable by assigning it a value

let Variable Keyword

  • A new keyword introduced with ES6
  • Variables declared with let can be changed or re-assigned, but not re-declared
let firstName;

let firstName = "Charmaine"

let firstName = "Michio" 
// Re-declaring firstName; this will throw an error

firstName = "Michio" // Re-assignment works

const Variable Keyword

  • A new keyword introduced with ES6
  • Used for values that should be constant or unchanged
  • Variables declared with const can not be re-assigned or re-declared
  • Variables declared with const have to initialized
const userName; // Not initialized; this will throw an error
  
const userName = "Jasmine";

userName = "Jackie" // Re-assignment; this will throw an error

var Variable Keyword

  • The pre-ES6 way of declaring variables
  • Not recommended to declare variables this way anymore
  • But still used in legacy code
var yourName = 'Alina';

Variable Naming

  • Starts with lowercase letters
  • Cannot be reserved JavaScript keywords
  • Can contain letters, numbers, _, $
  • Case sensitive
  • Should be descriptive
  • Best practice to write variables with camelCase syntax

Strings

A string is any sequence of alphanumeric characters, spacing and symbols

let randomLetters = "helll...  ///o@33!%%l"

Strings

  • Strings are characterized by opening and closing single or double quote marks
  • Opening and closing marks must be of the same type
let randomLetters = "helll...  ///o@33!%%l"

let greeting = 'Hello World'

Escaping Strings

If our string variables contain quotes or the backslash character, we will need to "escape" them

let sentence = "I waved to them: "Hello""

let sentence2 = 'What's your name?'

let sentence3 = 'I want to insert a backslash \ here.'

Escaping Strings

The Fix: Adding the backslash escape character

let sentence = "I waved to them: \"Hello\""

let sentence2 = 'What\'s your name?'

let sentence3 = 'I want to insert a backslash \\ here.'

String Concatenation

Strings can be combined using the + operator

let fullName = "John" + "Wick"

console.log(fullName)

Note: Empty or negative space is also considered a string

String Concatenation

String variables can also be combined or concatenated

let firstName = "John"
let lastName = "Wick"
console.log(firstName + lastName)

//or

let fullName = firstName + lastName;
console.log(fullName)

String Interpolation

  • A newer or ES6 way to combine strings is with template literals
  • JavaScript expressions can be embedded into a string statement

String Interpolation

  • The marker for template literals is an opening and closing backtick ` `and an expression ${someVariable}
let firstName = 'Beverly'

let lastName = 'Williams'

let fullName = `${firstName} ${lastName}`

console.log(fullName)
