A Gentle Introduction to ES6+

Variable Declaration

ES6 introduced two new keywords to declare variables

let and const

let Variable Keyword

  • With let, variables can be changed or re-assigned
  • Variables cannot be re-declared
let firstName;

let firstName = "Charmaine" // This will throw an error

firstName = "Charmaine" // Initializing the variable

firstName = "Michio" // Re-assigning the variable

let (MDN)

const Variable Keyword

  • Used for values that should be constant or unchanged
  • With const, a variable cannot be re-assigned or re-declared
  • Variables declared with const have to initialized
const userName; // Not initialized; this will throw an error
  
const userName = "Jasmine";

userName = "Jackie" // Re-assignment; this will throw an error

const (MDN)

var Variable Keyword

  • The pre-ES6 way of declaring variables
  • Not recommended to declare variables this way anymore
  • But still used in legacy code
var yourName = 'Alina';

What else is unique about
const and let?

Scope

Variables declared with const and let are block-scoped

Scope

Scope is how JavaScript manages the ability to use or access declared variables

There are three types of scope:

  • Global
  • Function
  • Block (Introduced in ES6)

A Simple Explanation of Scope in JavaScript

Global Scope

A variable declared in the global scope can be used anywhere in the code 

let year = 2022; // Variable in global scope

// Used anywhere in the code
console.log (year)

// Used in a function
function printYear(){
  console.log(year)
}
printYear()
  
// Used in a block
if(true){
  console.log(year)
}

Function Scope

A variable declared inside a function can only be used in that same function 

function printNewYear(){

  var y2k = 2000;

  console.log(y2k);

}

printNewYear() // Output: 2000

console.log(y2k) // Error

Block Scope

A variable declared inside of a block { }, for example, an if or else block, can only be used inside that block 

let age = 20 

if(age === 20){
  
  let greeting = `Happy Birthday!`

  console.log(greeting) // Output: Happy Birthday!

}

console.log(greeting) // Error

String Interpolation or

Template Literals

String Interpolation

  • The ES6 way of combining strings
  • Also called template literals
  • More efficient than concatenating
let firstName = 'Beverly'
let lastName = 'Williams'

// ES5 - Concatenation
let fullName1 = firstName + ' ' + lastName
console.log(fullName1)

// ES6
let fullName2 = `${firstName} ${lastName}`
console.log(fullName2)

String Interpolation in JavaScript

String Interpolation

  • Wrap the entire string in backticks
  • Use ${variable_name} to denote and enclose variables
  • JavaScript expressions can be embedded into a string statement
let firstName = 'Beverly'

let lastName = 'Williams'

let currentYear = 2022

let birthYear = 1948

let statement = `My first name is ${firstName}; last name ${lastName}, and I am ${currentYear - birthYear} years old.`

console.log(statement)

Arrow Functions

Arrow Functions

Introduced in ES6, arrow functions provide an alternative and more concise way to write function syntax

An Introduction to JavaScript Arrow Functions

Arrow Functions

// Standard function syntax
function greeting(name){
  return `Hello ${name}`
}

console.log(greeting('Kemi'))


// Same function;
// Using an arrow function expression
const greeting = (name) => {
  return `Hello ${name}`
}

console.log(greeting('Keisha'))

Arrow Function Syntax

const functionName = () => {

// Code to run

}

functionName()

Arrow Function Shorthands

  • An arrow function with a single parameter can omit the parentheses around the parameter
  • An arrow function with a single or simple expression can omit the curly brackets and the return keyword 
function greetFriend (firstName) {

  return `Hey ${firstName}`

}
  
// Arrow Function Shorthand

const greetFriend = firstName => `Hey ${firstName}`

for of Loop

for of Loop

  • The for of loop was introduced in ES6
  • It can be used on any data type that's countainable, such as strings and arrays
  • It is written differently than the standard for loop

Introduction to the JavaScript for...of Loop

for of Loop Syntax

for (let item of iterable) {
  
  // ...
  // usually returning or doing something with 'variable'

}

iterable is the element that's being looped

item is each item in the looped element

Comparison: Standard for Loop

Output each character of a string

let famousName = "oprah"

for(let i = 0; i < famousName.length; i++){
  
  console.log(famousName[i])

}

Comparison: for of Loop

Output each character of a string

let famousName = "oprah"

for(let char of famousName){
  
  console.log(char)

}
