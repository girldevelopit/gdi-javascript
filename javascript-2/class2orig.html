Array Methods:
Higher Order Functions

The findIndex method is a type of function known as a Higher Order Function

A higher order function accepts another function as a parameter or can return a function as an output

Higher Order Function

  • To use findIndex to get the index of 'Manilla', we need to pass it another function as a parameter
  • This second function creates a testing condition that will be applied to each element in the array 
let asianCapitals = ['Seoul', 'Dhaka', 'Manilla', 'Bangkok']

let result = asianCapitals.findIndex(function(city,index){
  if(city === 'Manilla'){
    return index
  }
})

console.log(result) // 2

Callback Function

A function that can be passed as a parameter is known as a callback function

Let's rewrite our code to illustrate a callback function getting passed to findIndex() 

let asianCapitals = ['Seoul', 'Dhaka', 'Manilla', 'Bangkok']

function findManilla(city){
  if(city === 'Manilla'){
    return true;
  }
}

let result = asianCapitals.findIndex(findManilla)

// findManilla() is a callback function because 
// it is passed as a parameter to findIndex()

console.log(result) // 2

Higher Order Functions
and Arrow Function Syntax

Arrow functions are often used to simplify how we write higher order functions 

let result = asianCapitals.findIndex(function(city,index){
  if(city === 'Manilla'){
    return index
  }
})
  
// Rewritten with arrow function syntax

let result = asianCapitals.findIndex((city, index) => {
  if(city === 'Manilla'){
    return index
  }
})

Array Iteration Methods

The findIndex method is one of several array methods that can iterate (loop) through an array and execute an action on each item in the array

These methods are all higher order functions because they require another function as a parameter

Array Iteration Methods

Array iterators loop or iterate through an array and perform an action on each item in the array

Array iterators require a second function to be passed as a parameter

This second function contains the action to be applied to each element

Common Array Iteration Methods

  • .map()
  • .forEach()
  • .filter()
  • .reduce()

  • .some()

How to Use Array Iteration Methods in JavaScript

.map()

Creates a new array that contains the results of a callback function that's been applied to every element in the original array 

const numbers = [1,2,3,4]

const doubleNumbers = numbers.map((num) => {
  return num * num
})

console.log(doubleNumbers) // Output: [1,4,9,16]

// Arrow syntax shorthand version:

// const doubleNumbers = numbers.map(num => num * num)

Array.prototype.map() - MDN

.filter()

Creates a new array with elements that pass the test condition of a callback function 

// Filter words that don't contain the string 'cat'
  
let catStrings = ['avocado', 'advocate', 'catty', 'catacomb']

let nonCatStrings = catStrings.filter((item) => {
  return !item.includes('cat')
})

console.log(nonCatStrings) 

// Arrow syntax shorthand version:

// const nonCatStrings = catStrings.filter(item => !item.includes('cat'))

Array.prototype.filter() - MDN

Return Values

While some iteration methods create a new array, others return elements from the original array or a boolean value; and one particular method, the forEach, does not return any value at all

.forEach()

Runs an action on every element in an array; unlike other methods, it does not return an actual value 

let ageList = [4,5,6,7]

const updatedAgeList = ageList.forEach( element => element + 5 ) 

console.log(updatedAgeList) // Returns undefined

.forEach() - MDN

.forEach()

The forEach behaves similarly to the for loop and the for of loop 

let ageList = [4,5,6,7]

ageList.forEach( element => console.log(element + 5) ) 
// Output: 9, 10, 11, 12

for(let i = 0; i  < ageList.length; i++){
  console.log( ageList[i] + 5)
}
// Output: 9, 10, 11, 12

for(let age of ageList){
  console.log(age + 5)
}
// Output: 9, 10, 11, 12

.forEach() - MDN

When to Use .forEach?

forEach can be an alternative to using a for loop or a for...of loop; all three iterate through every element in an array

What About Other Methods?

A few tips:

  • Use .map to change items in an array
  • Use .filter to select a subset of an array
  • Use .find to select one item in an array
