Strings

A string is any sequence of alphanumeric characters, spacing and symbols

let randomLetters = "helll...  ///o@33!%%l"

Strings

  • Strings are characterized by opening and closing single or double quote marks
  • Opening and closing marks must be of the same type
let randomLetters = "helll...  ///o@33!%%l"

let greeting = 'Hello World'

Escaping Strings

If our string variables contain quotes or the backslash character, we will need to "escape" them

let sentence = "I waved to them: "Hello""

let sentence2 = 'What's your name?'

let sentence3 = 'I want to insert a backslash \ here.'

Escaping Strings

The Fix: Adding the backslash escape character

let sentence = "I waved to them: \"Hello\""

let sentence2 = 'What\'s your name?'

let sentence3 = 'I want to insert a backslash \\ here.'

String Concatenation

Strings can be combined using the + operator

let fullName = "John" + "Wick"

console.log(fullName)

Note: Empty or negative space is also considered a string

String Concatenation

String variables can also be combined or concatenated

let firstName = "John"

let lastName = "Wick"

console.log(firstName + lastName)

//or

let fullName = firstName + lastName;

console.log(fullName)

String Interpolation

  • A newer or ES6 way to combine strings is with template literals
  • JavaScript expressions can be embedded into a string statement

String Interpolation

  • The marker for template literals is an opening and closing backtick ` `and an expression ${someVariable}
let firstName = 'Beverly'

let lastName = 'Williams'

let fullName = `${firstName} ${lastName}`

console.log(fullName)

Concatenate or Interpolate?

Interpolation is more efficient.

String.length

Use the length property to find the length of a string

let badPassword = 'mydob'

console.log(badPassword.length) // 5

String: length (MDN)

Zero Index

  • JavaScript is a zero-indexed language
  • We count from 0, not 1
  • Every character in a string has an index value--a number corresponding to its position in the string

Accessing String Characters

Type the string variable and use the bracket notation []:

let badPassword = 'mydob'

console.log( badPassword[0] ) // 'm'

console.log( badPassword[1] ) // 'y'

console.log( badPassword[2] ) // 'd'

console.log( badPassword[3] ) // 'o'

console.log( badPassword[4] ) // 'b'

string.indexOf()

We can search for characters

let shortName = 'Mary'

console.log(shortName.indexOf('a')) // 1
// Returns index of first occurrence of 'a'


let song = "99 bottles of wine on the wall..."

console.log(song.indexOf('beer')) // -1
// Returns -1 if a character doesn't exist

String: indexOf (MDN)

string.toUpperCase()

We can change case

let greeting = 'hello world'

let shouting = greeting.toUpperCase()

console.log(shouting) // HELLO WORLD

String: toUpperCase (MDN)

string.replace()

We can replace characters

let statement = 'I Love JavaScript!'

// Replace words
statement = statement.replace('JavaScript', 'Python')

console.log(statement)


// Replace characters
statement = statement.replace('o', '0')

console.log(statement)

String: replace (MDN)
