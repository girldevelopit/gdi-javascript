Functions

  • A function is a block of code with instructions to perform a particular task
  • Functions are reusable--they can be called multiple times to perform the same task in different parts of a codebase
  • Functions make our code organized and more efficient

JavaScript Functions (W3Schools)

Functions (MDN)

Function Declaration

We declare a function with the keyword function, a function name, a pair of parentheses and curly braces

// The syntax of a function declaration:

function funtionName(){

  // instructions to carry out 

}

Function Declaration

function sayGreeting(){

  console.log('Hello everyone!')

}

Function Call

But declaring a function is not enough. It then needs to be invoked or called or run. These are all ways of saying we want to use or execute the function.

// The syntax of a function call:

functionName()

Declaring and Calling a Function

function sayGreeting(){

  console.log('Hello everyone!')

}

Function Parameters

  • Functions can take (optional) input values known as parameters
  • A parameter is a placeholder for a value that the function will use when called
// Syntax of using or passing a parameter:

function friendlyGreeting( anyStringName ){

  console.log(`Hi ${anyStringName}`)

}

friendlyGreeting('') 
// A string value goes into the function call

Function Argument

  • An argument is the actual value passed to the function to use (when invoked) instead of the placeholder
  • Arguments can be any data type: Strings, Numbers, Boolean, Arrays, etc.
// Syntax of using an argument:

function friendlyGreeting( anyName ){

  console.log(`Hi ${anyName}`)

}

friendlyGreeting('Tamara')

// We've passed an argument, the string 
// 'Tamara' to the function to use

Function Argument

  • We can now reuse the function multiple times with different arguments 
// Syntax of using an argument:

function friendlyGreeting( anyName ){

  console.log(`Hi ${anyName}`)

}

friendlyGreeting('Tamara')
friendlyGreeting('Jamal')
friendlyGreeting('Cindy')

Variables as Arguments

Since variables are values, they can be passed to functions as arguments

let firstNumber = 3
let secondNumber = 7
let thirdNumber = 8


function addNumbers(num1, num2, num3){

  console.log(num1 + num2 + num3)

}

addNumbers(firstNumber, secondNumber, thirdNumber)

// This adds 3 + 7 + 8

Return Statements

We've been using the console.log method to output our JavaScript code, but the console is a temporary environment;

We want the results of our code to be stored and available for use whenever it is needed

Return Statement

A return statement sets the value a function returns when the function is called

function addTwoNumbers (num1, num2){

  let sum = num1 + num2;

  return sum;

}

Return and Function Return Values (MDN)

Return Statement

A return statement also ends a function's execution

function addTwoNumbers (num1, num2){

  let sum = num1 + num2

  return sum

  console.log('Hello') // This will not run

}

Return Statement

A function without a return statement
will be undefined

function addTwoNumbers (num1, num2){

  let sum = num1 + num2

}

console.log(addTwoNumbers(4,5)) // Undefined

Return Value

A function's return value can be
saved and used at any time

function addTwoNumbers (num1, num2){

  let sum = num1 + num2;

  return sum;

}

let calculateFiveAndTen = addTwoNumbers(5,10)

console.log(calculateFiveAndTen)

console.log(`5 plus 10 equals ${calculateFiveAndTen}`)
